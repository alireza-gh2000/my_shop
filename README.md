# فروشگاه اینترنتی عطر و ادکلن

این پروژه یک فروشگاه اینترنتی ساده با استفاده از فریم‌ورک Django می‌باشد
که در قالب پروژه کارشناسی رشته مهندسی نرم‌افزار پیاده‌سازی شده است.

## امکانات پروژه
- صفحه محصولات
- ثبت‌نام و ورود کاربران
- سبد خرید
- ثبت سفارش
- مشاهده سفارش‌های کاربر
- پنل مدیریت Django برای مدیریت محصولات و سفارش‌ها

## فناوری‌های استفاده‌شده
- Python
- Django
- HTML / CSS
- SQLite

## توضیح
داده‌های موجود در پروژه (محصولات و سفارش‌ها) صرفاً داده‌های تستی
جهت نمایش قابلیت‌های سیستم می‌باشند.
